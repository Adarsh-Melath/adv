## 1.JDBC Fundamentals

a) Set up JDBC in a Java project.

b) Perform SQL queries (SELECT, INSERT, UPDATE, DELETE) using JDBC.

c) Use Prepared Statements for safe and efficient query execution.

d) Understand and manage transactions (commit, rollback).

## 2.Introduction to ORM (Object Relational Mapping)

a) Understand the concept and benefits of ORM in enterprise applications.

b) Learn about Hibernate and Java Persistence API (JPA).

## 3.Working with Hibernate and JPA

a) Perform CRUD operations using Spring Data JPA Repository.

b) Define and manage Entity relationships:

     > One-to-One
     > One-to-Many
     > Many-to-Many

c) Understand Lazy vs Eager Loading and their implications on performance

## 4.Mini Project: Web Application with Admin Panel

a) User Module (Frontend + Backend)

> Build a signup page to register users (store data in DB).
> Build a login page with proper validation.
> Create a home/dashboard page for authenticated users.

b) Admin Module

> Create a secure admin login page with validation.
> Admin should be able to: - View the list of registered users. - Search users by name/email. - Create, Update, and Delete user records.

c) Application Behavior & Security

> Implement session and cookie handling to manage user sessions securely.
> Ensure role-based access (user vs admin).
> Handle form validation, error messages, and redirection properly.
